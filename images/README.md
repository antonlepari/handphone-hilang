## Panduan Kontribusi Gambar

Jika ingin menambahkan screenshot:
- Format: PNG, maksimal 1 MB per file
- Resolusi: 1920×1080 atau lebih rendah
- Hapus informasi pribadi sebelum mengunggah (nama akun, nomor telepon)
- Gunakan nama file yang deskriptif seperti tertera di tabel di atas
