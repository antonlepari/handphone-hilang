# Alat Keamanan Mobile

> Terakhir Diperbarui: 2026-06-01

## Ikhtisar

Bagian ini mendokumentasikan alat-alat open-source dan komersial yang relevan untuk keamanan mobile, respons insiden, forensik digital, dan analisis malware mobile.

---

## Kategori Alat

| Kategori | Deskripsi |
|---|---|
| **Forensik & Analisis** | Ekstraksi data, analisis artefak, deteksi spyware |
| **Pengujian Keamanan** | Analisis statis/dinamis aplikasi mobile |
| **Pelacakan & Anti-Pencurian** | Pelacakan lokasi, kunci jarak jauh, penghapusan |
| **Manajemen Perangkat** | MDM, konfigurasi, manajemen kebijakan |
| **Instrumentasi & Debug** | Hooking, patching, analisis runtime |
